== Installation ==

1. Upload the plugin files to `/wp-content/plugins/smart-chatbot/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your OpenAI API key to `wp-config.php`:
   `define('OPENAI_API_KEY', 'sk-your-api-key-here');`
4. Navigate to 'Smart Chatbot' in your admin menu
5. Configure your company information and preferences
6. The chatbot will appear on all pages automatically

== Frequently Asked Questions ==

= Do I need an OpenAI API key? =

Yes, you'll need an OpenAI API key to use this plugin. You can get one at https://platform.openai.com/api-keys

= Can I customize the chatbot appearance? =

Yes! The admin panel allows you to customize colors, position, and behavior. Advanced users can also modify the CSS.

= Is it multilingual? =

Yes, the chatbot supports multiple languages including UK English (default), US English, Spanish, French, and German.

= What about GDPR compliance? =

The plugin stores minimal data and includes privacy controls. Always ensure you have proper consent mechanisms in place.

= Can I track chatbot performance? =

Yes! The built-in analytics dashboard shows conversations, conversion rates, user satisfaction, and more.
